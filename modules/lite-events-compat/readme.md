# RxJS Lite DOM Events Compat Module #

This project provides Reactive Extensions for JavaScript (RxJS) bindings for HTML DOM objects to abstract over the event binding using RxJS Lite Compat for older browsers.  

## Details ##

Files:
- [`rx.lite.dom.events.compat.js`](https://github.com/Reactive-Extensions/RxJS-DOM/blob/master/modules/lite-events-compat/rx.lite.dom.events.compat.js)

NPM Packages:
- [`rx-lite-dom-events-compat`](https://www.npmjs.com/package/rx-lite-dom-events-compat)
